# Usage
```sh
# Install
$ pnpm i

# Running the app
$ pnpm start      # development
$ pnpm start:dev  # watch mode
$ pnpm start:prod # production mode

# Running tests
$ pnpm test
$ pnpm test:e2e # e2e tests
$ pnpm test:cov # test coverage
```
